# 💪 Workout Tracker Application

A premium, modern React + Vite web application to plan, log, and analyze your workouts.

---

## 🚀 How to Run the App 

We have created a double-clickable shortcut file to run this app on Windows without writing any commands!

### Option 1: Double-Click Shortcut (Sabse Aasan Tarika)
1. Go to the project folder (`Workout Tracker`).
2. Double-click the file **`Start-Workout-Tracker.bat`**.
3. It will:
   - Automatically install the required files (only the first time).
   - Start the local server.
   - Automatically open the app in your web browser!

---

### Option 2: Run via Terminal (Manually)
If you prefer running commands, open your terminal (PowerShell or Command Prompt) in the project folder and run:

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Start Dev Server:**
   ```bash
   npm run dev
   ```
3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## 🐙 How to Add & Push to GitHub (GitHub me upload kaise kare)

We have already initialized Git, configured your remote repository, and made the initial commit. Follow these simple steps to upload it to your GitHub:

### Step 1: Create a repository on GitHub
1. Go to [GitHub.com](https://github.com/) and log in to your account (`naitik2424`).
2. Click the **"New"** button (or go to `https://github.com/new`) to create a new repository.
3. Keep the repository name exactly: **`workout-tracker`**
4. Do **NOT** initialize it with a README, `.gitignore`, or License (as we already have them in our project).
5. Click **Create repository**.

### Step 2: Push your code to GitHub
Once you have created the repository, run the following command in your VS Code terminal or PowerShell:

```bash
git push -u origin main
```

*(Note: Git might open a window asking you to sign in to your GitHub account to complete the upload securely. Just sign in, and your code will be uploaded!)*
